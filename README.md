### 使用说明

```
# 克隆项目
git clone https://gitee.com/lucky_ck/vue3-big-screen.git

# 进入项目
cd vue3-big-screen

# 安装依赖
npm install

# 运行项目
npm run dev

# 打包发布
npm run build
```
